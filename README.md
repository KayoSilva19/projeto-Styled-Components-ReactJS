### 🎲 Rodando o Front 

```bash

    # Clone este repositório  
    $ git clone <https://github.com/KayoSilva19/projeto-Styled-Components-ReactJS>

    # Acesse a pasta do projeto no terminal/cmd
    $ cd projeto-Styled-Components-ReactJS

    # Instalando Dependências
    $ npm install

    # Rodando o Projeto
    $ npm start

```
